#  Swiggy Sales Dashboard (Excel Project)

##  Overview
This project is an interactive Swiggy Sales Dashboard built using Microsoft Excel. It provides insights into sales, ratings, orders, and customer behavior across different cities and states.

---

##  Dashboard Features

###  Key Metrics
- Total Sales: ₹2.50M  
- Average Rating: 4.34  
- Average Order Value: 0.36M  
- Rating Count: 10.29K  
- Total Orders: 243K  

---

###  Visual Analysis
- Monthly Sales Trend
- Daily Sales Trend
- Weekly Sales Trend
- Sales by Food Type (Veg vs Non-Veg)
- Sales by State (India Map)
- Top 5 Cities by Sales
- Quarterly Performance Table

---

###  Interactive Filters
- Month Filter  
- Restaurant Name Filter  
- Category (Cuisine Type) Filter

##  How to Use
1. Download the Excel file  
2. Open Dashboard Sheet  
3. Use slicers to filter data  
4. Analyze charts and insights  

---

##  Key Insights
- Non-Veg contributes ~65% of total sales  
- Bengaluru is the top city  
- Sales are stable across weeks  
- High average rating (4.34)
