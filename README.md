##  Project Overview
This project is an interactive **Swiggy Sales Dashboard** built using Microsoft Excel to analyze food delivery data.

The goal of this project is to extract meaningful insights from raw data and support **data-driven business decisions**.

---

##  Business Problem
Food delivery platforms generate large volumes of data, but without proper analysis:

- Sales trends are unclear  
- Customer preferences are unknown  
- High-performing locations are difficult to identify  

 This dashboard solves these problems by providing clear and actionable insights.

---

##  Dashboard Features

###  Key Metrics
- Total Sales: ₹2.50M  
- Average Rating: 4.34  
- Average Order Value: 0.36M  
- Rating Count: 10.29K  
- Total Orders: 243K  

###  Analysis Included
- Monthly Sales Trend
- Daily & Weekly Trends
- Sales by Food Type
- Sales by State
- Top Cities Performance

##  Key Insights
- Non-Veg contributes ~65% of total sales
- Bengaluru is the top-performing city
- Consistent weekly sales trend
- High customer satisfaction (Rating: 4.34)

##  Business Recommendations
